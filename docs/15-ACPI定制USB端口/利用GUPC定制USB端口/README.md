# 利用GUPC方法定制USB端口

## 描述

- 本方法通过修改 ACPI 中的GUPC方法实现USB端口的定制。
- 不需要Drop掉任何SSDT
- 本方法献给爱好者们。

### 解释

- 利用序列化的**GUPC方法**，通过修改该方法来实现USB端口的定制
